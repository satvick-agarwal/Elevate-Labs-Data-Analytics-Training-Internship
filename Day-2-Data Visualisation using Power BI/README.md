# 📊 Superstore Sales & Profit Analysis Dashboard

An interactive Power BI dashboard built using the Sample Superstore dataset to analyze sales performance, profitability, regional trends, customer segments, and product insights. The project demonstrates the complete business intelligence workflow—from data validation and preparation to dashboard design and business storytelling.

---

## 📌 Project Objective

Develop an executive-level dashboard that enables stakeholders to monitor business performance, identify profitability drivers, and uncover actionable insights for strategic decision-making.

---

## 📂 Dataset

**Sample Superstore Dataset**

The dataset contains retail transaction records including:
- Orders
- Customers
- Products
- Sales
- Profit
- Quantity
- Discount
- Shipping Information
- Geographic Details

---

## 🛠 Tools Used

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)

---

## 🧹 Data Preparation

The dataset was validated and prepared using **Power Query**.

Performed the following checks:

- Verified data types
- Checked for missing values
- Checked duplicate records
- Validated Order IDs
- Verified categorical values
- Performed data quality assessment

No major inconsistencies were found in the dataset.

## 📊 Key Business Insights

### Sales Dashboard

- Technology is the highest revenue-generating category.
- Consumer segment contributes over 50% of total sales.
- West region records the highest sales.
- Standard Class is the most frequently used shipping mode.
- Sales exhibit a consistent upward trend from 2014 to 2017.

### Profit Dashboard

- Technology is the most profitable category.
- Tables is the primary loss-making sub-category.
- A small number of products contribute disproportionately to overall losses.
- Higher discounts generally reduce profitability.
- Profitability varies significantly across different states.
